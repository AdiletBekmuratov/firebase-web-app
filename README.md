# Firebase Web App

## Features of Web App
* Fully responsive
* Connected with real API
* Live Search system
* Cart system

# Getting started
1) Clone this repo
2) Follow the link and make necessary steps in [Firebase Documentation](firebase.google.com/docs/web/)
3) Go to `js/firebase_config.js` and add your credentials

## Application Structure
* `images/` - all pictures used in project. Thanks for [Unsplash](https://unsplash.com)
* `js/` - all necessary javascript and jquery codes
* `templates/` - blocks of html code used in js files

For database used **Firebase** realtime database
